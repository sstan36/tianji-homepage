# Tianji Homepage (Static)

This repo contains the static marketing site for **Tianji, Secrets of Destiny**.

## Pages

- `index.html` — Home (includes a simple “Marketing News” section)
- `about.html`
- `support.html`
- `privacy.html` — links out to the currently-hosted privacy policy

## Local preview

Open `index.html` directly in a browser, or run a simple static server (optional):

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Before publishing

- Replace `support@yourdomain.com` in `support.html`
- Ensure the App Store links point to `https://apps.apple.com/us/app/tianji-bazi-destiny/id6761753338`
- Replace the placeholder `assets/*.svg` with real screenshots
